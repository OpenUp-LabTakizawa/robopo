# ログインの仕方

管理者・採点者として操作するためには、ユーザー名とパスワードでログインが必要です。観戦ページだけを見る場合はログインは不要です。

:::tip[この画面を開く]

👉 [https://robopo.caravan-kidstec.com](https://robopo.caravan-kidstec.com)

:::

## 手順

1. トップページ右上の **「ログイン」** ボタンを押します。

2. ダイアログが開くので、ユーザー名とパスワードを入力し **「ログイン」** を押します。

   <BaseImage src="/screens/02-getting-started/login-dialog.webp" alt="ログインダイアログ" width={500} />

3. ログインに成功すると、次の画面が表示されます。
   - **管理者アカウント**の場合: 3つのカード（採点・大会管理・設定）が並んだダッシュボード
   - **採点者アカウント**の場合: 採点開始用の選択フォームのみ

<BaseImage src="/movies/01-login-to-dashboard.webp" alt="ログインフロー" width={700} />

:::info[ログインダイアログの操作]

- ダイアログの **「戻る」** ボタンまたは背景クリックで、ログインせずに閉じることもできます
- ユーザー名またはパスワードが空の場合、**ログインボタンは押せません**（無効化）
- 入力ミスで失敗した場合は、ダイアログ内にエラーメッセージが表示されます

:::

## ログアウト

画面右上のユーザー名の横にある **「ログアウト」** ボタンを押すと、ログアウトして未ログイン状態に戻ります。

## アカウントを忘れたら

- **ユーザー名を忘れた場合**: 他の管理者に [管理者一覧](/docs/admin/admin-users) ページで確認してもらってください
- **パスワードを忘れた場合**: パスワードの復元はできません。他の管理者に新しいパスワードをセットしてもらう必要があります

:::caution

1人目の管理者しかいない状態でパスワードを忘れると **誰もログインできなくなる** ため、運用前に必ず **複数の管理者アカウント** を作成しておいてください。

:::
