# 管理者・採点者・観客の違い

ROBOPOには3種類のユーザーがいます。それぞれできることと、ログインの要否が異なります。

## 一覧表

| 役割 | ログイン | できること |
| --- | --- | --- |
| **管理者** | 必要 | すべての機能（大会運営・採点・集計・ユーザー管理） |
| **採点者** | 必要 | 担当大会のコースに対する採点のみ |
| **観客** | 不要 | 観戦ページでライブランキングを閲覧 |

## 管理者

大会運営の中心となる担当者です。ログインすると、**すべての機能** が利用できます。

- 大会・コース・選手・採点者の登録と編集
- 採点の実施
- 集計結果の閲覧・印刷
- 他の管理者アカウントの追加・削除

管理者のログイン後はダッシュボード（採点 / 大会管理 / 設定 の3カード）が表示されます。詳しくは [ダッシュボード画面](/docs/admin/dashboard) を参照してください。

## 採点者

採点画面の操作のみを担当します。試合中にタブレット・スマホで選手の得点を記録します。

- **自分のユーザー名・パスワードでログイン** して利用します
- ログイン後は **管理メニューは表示されず、採点画面だけ** が使えます
- 自分が担当する大会のコースに対してのみ採点可能
- 採点結果には「誰が採点したか」が自動的に記録されます

:::info

採点者のアカウントはあらかじめ管理者が [採点者の登録](/docs/admin/judge) ページから作成します。当日はそのユーザー名・パスワードで採点者自身がログインします。

:::

## 観客（ログイン不要）

会場の観客・保護者は、ログインせずに **観戦ページ** からライブランキングを確認できます。

- 👉 [https://robopo.caravan-kidstec.com/spectator](https://robopo.caravan-kidstec.com/spectator)
- 端末を選ばず閲覧でき、スマホ表示に最適化されています
- 開催中の大会だけが選択肢に表示されます
- 順位は数十秒ごとに自動更新されます

詳しくは [ライブランキングの見方](/docs/spectator/spectator) を参照してください。

## ユーザーの種類を判定する仕組み

ログインしたユーザーが採点者か管理者かは、以下のルールで自動判定されます。

- **採点者テーブルに紐付いているユーザー** → 採点者として扱う
- **それ以外のログイン済みユーザー** → 管理者として扱う

つまり、採点者は [採点者の登録](/docs/admin/judge) 画面で作成すると自動的に採点者権限になり、[管理者アカウント管理](/docs/admin/admin-users) 画面で作成すると管理者権限になります。
