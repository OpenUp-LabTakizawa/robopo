# ダッシュボード画面

管理者としてログインすると、トップページに3つのカードが並んだダッシュボードが表示されます。ここから採点・管理・設定のすべての機能に1クリックでアクセスできます。

:::tip[この画面を開く]

👉 [https://robopo.caravan-kidstec.com](https://robopo.caravan-kidstec.com)

:::

<BaseImage src="/screens/03-admin/dashboard.webp" alt="管理者ダッシュボード" width={800} />

:::info[表示されるのは管理者だけ]

このダッシュボードは **管理者アカウントでログインしたとき** にのみ表示されます。採点者アカウントでログインした場合は「採点」カードの部分だけが単体で表示されます（大会管理・設定メニューは見えません）。

:::

## 1. 採点カード

ダッシュボードの一番目立つ位置にある主要エリアです。**大会・コース・採点者・選手** の4つを選んで採点画面に進みます。

| 項目 | 内容 |
| --- | --- |
| 大会を選択 | **開催中** の大会のみ表示。開催中が1つだけなら自動で選ばれます |
| コースを選択 | 選んだ大会に割り当てられたコースが表示されます |
| 採点者を選択 | 選んだ大会の担当採点者を検索ボックス付きで選択 |
| 選手を選択 | 選んだ大会に参加する選手を検索ボックス付きで選択 |

すべて選ぶと **「採点を開始」** ボタンが有効になり、押すと採点画面へ遷移します。

:::tip[前回の選択を自動で覚えてくれます]

選んだ大会・コース・採点者は **ブラウザに保存** されており、次回同じ画面を開いた時に前回の選択が自動で復元されます。大会中の連続採点でも毎回選び直す必要がありません（選手だけは毎回選び直しになります）。

:::

:::info[開催中の大会とは]

開催中と判定されるのは、大会の **開催日時** が過ぎていて、**終了日時** に達していない大会です。過去の大会や未来の大会はここに出てきません（集計結果画面からは確認できます）。

:::

## 2. 大会管理カード

大会運営のためのページへのリンク集です。

- **大会一覧** — 大会の作成・編集・削除
- **コース一覧** — コースの作成・編集・削除
- **選手一覧** — 選手の登録・編集・削除
- **採点者一覧** — 採点者の登録・編集・削除
- **集計結果** — 大会ごとの成績確認と印刷

## 3. 設定カード

システム全体の設定ページです。

- **管理者一覧** — 管理者アカウントの追加・削除・パスワード変更

## 画面右上のヘッダー

| 要素 | 内容 |
| --- | --- |
| ユーザー名表示 | 現在ログイン中のユーザー名 |
| ログアウトボタン | クリックでログアウトして未ログイン状態に戻ります |
| メニュー（≡） | サイドメニューを開き、どのページからでも全ナビゲーションにアクセス可能 |

## 未ログイン時の画面

ログインしていない状態でトップページを開くと、ダッシュボードは表示されず、代わりに **「ログイン」ボタン** と **「LIVE観戦」ボタン** が中央に表示されます。観戦ページはログイン不要で見られます。
