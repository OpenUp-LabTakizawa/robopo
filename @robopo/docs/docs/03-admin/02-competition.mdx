# 大会の作成・編集

大会（Competition）は、ROBOPOの管理単位の中で最も大きな単位です。1つの大会に対してコース・選手・採点者を紐づけて、競技を運営します。

:::tip[この画面を開く]

👉 [https://robopo.caravan-kidstec.com/competition](https://robopo.caravan-kidstec.com/competition)

:::

## 大会一覧

<BaseImage src="/screens/03-admin/competition-list.webp" alt="大会一覧" width={900} />

| 項目 | 説明 |
| --- | --- |
| ID | システムが自動採番する連番 |
| 名前 | 大会名 |
| コース | 登録されているコース名（複数あればまとめて表示） |
| 開催日時 | 開催開始の日時（分単位） |
| 終了日時 | 大会終了の日時（分単位） |
| マスク | 選手名マスク機能 ON / OFF |
| マスク時間 | マスクON時、終了前N分からマスクを開始 |
| 説明 | 自由記述メモ |

## 検索・絞り込み・並び替え

- **検索ボックス**: 名前・説明で検索
- **状態フィルタ**: すべて / 開催前 / 開催中 / 終了済 / 未設定 の5択
  - 未設定 = 開催日時／終了日時が未入力の大会
- **並び替え**: ID / 名前 / 開催日 / 終了日 のいずれかで昇順・降順切替

## 新規作成

1. **「新規作成」** ボタンを押します。
2. ダイアログが開くので、以下の項目を入力します。

   <BaseImage src="/screens/03-admin/competition-create.webp" alt="大会作成ダイアログ" width={700} />

   | 項目 | 必須 | 説明 |
   | --- | --- | --- |
   | 名前 | ✅ | 大会名 |
   | 説明 | - | 自由記述メモ |
   | 開催日時 | - | 分単位で指定（`datetime-local` 入力） |
   | 終了日時 | - | 分単位で指定 |
   | 終了間際に選手名をマスクする | - | チェックで有効化 |
   | 終了前N分からマスク | - | マスクON時のみ表示。1〜999分、既定30分 |
   | コース | - | 登録済みコースから複数選択可。 **THE一本橋・センサーコース** があれば初期選択される |

3. **「作成」** を押すと大会が追加されます。

:::tip[選手名マスクとは？]

大会終了間際になると、観戦ページに表示されている **選手名を自動的に「???」に隠す** 機能です。

- **目的**: 表彰式の直前まで「誰が優勝するか分からない」というワクワク感を会場に演出する
- **例**: 大会終了30分前からマスクONにすると、そこからは順位と得点だけが見えて、名前は伏せられます
- **不要な場合**: OFF にしておけば、最後まで通常通り選手名が表示されます

詳しくは [マスク機能について](/docs/spectator/mask) を参照してください。

:::

:::caution[日時のバリデーション]

**開催日時は終了日時より前** でなければなりません。逆転していると赤いエラーメッセージが表示されて保存できません。片方だけ入力することは可能です（例: 終了日未定の場合）。

:::

## 編集

1. 一覧で対象の大会を **1つだけ** 選択します（チェックボックス）。
2. **「編集」** ボタンを押します（1件選択時のみ有効）。
3. ダイアログで項目を更新し **「保存」**。

:::info

コースの紐付けも編集ダイアログから変更できます。チェックを外せば紐付け解除、チェックを追加すれば新規紐付けです。

:::

## 削除

- チェックボックスで **複数選択** してから **「削除」** ボタンで一括削除
- 確認ダイアログに削除対象の大会名リストが表示されます

:::caution

削除した大会はもとに戻せません。採点データも同時に失われるため、削除は十分注意して行ってください。

:::

## 大会の「状態」について

ROBOPOは大会の日時設定から自動的に状態を判定します。

| 状態 | 条件 |
| --- | --- |
| 開催前 | 現在時刻 < 開催日時 |
| 開催中 | 開催日時 ≤ 現在時刻 ≤ 終了日時 |
| 終了済 | 現在時刻 > 終了日時 |
| 未設定 | 開催日時・終了日時の少なくとも片方が未入力 |

- ダッシュボードの採点カードには **開催中** の大会のみ表示されます
- 集計結果・観戦ページでも **開催中または終了済** のみが対象になります
- **未設定**の大会は一覧で管理できますが、ダッシュボードからは採点できません
