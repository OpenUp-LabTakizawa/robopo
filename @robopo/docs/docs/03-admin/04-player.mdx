# 選手の登録

選手（Player）は、大会に参加する子どもたちです。氏名・ふりがな・ゼッケン番号・参加大会などを登録します。

:::tip[この画面を開く]

👉 [https://robopo.caravan-kidstec.com/player](https://robopo.caravan-kidstec.com/player)

:::

## 選手一覧

<BaseImage src="/screens/03-admin/player-list.webp" alt="選手一覧" width={900} />

| 項目 | 説明 |
| --- | --- |
| ID | システムが自動採番する連番 |
| ゼッケン番号 | 選手を識別する番号（任意・文字列でも可） |
| ふりがな | ひらがな／カタカナ。並び替えや印刷時の並びに使われます |
| 名前 | 選手名（必須・重複不可） |
| 参加大会 | この選手が参加する大会（複数選択可） |
| 備考 | 自由記述メモ（アレルギー情報・保護者連絡先・特記事項など運営メモ用） |

## 検索・絞り込み・並び替え

- **検索ボックス**: 名前・ふりがな・ゼッケン番号・**備考** のいずれかに一致する選手を絞り込み
- **大会フィルタ**: プルダウンから1つの大会を選ぶと、その大会に参加する選手だけを表示
- **並び替え**: ID / 名前 / ふりがな / ゼッケン番号 のいずれかをキーに、昇順／降順で切替可能
  - 名前・ふりがなは A→Z / Z→A、ゼッケン番号は数値として比較されます

## 新規作成

1. **「新規作成」** を押します。
2. ダイアログで以下を入力します（**名前以外は任意**）。

   | 入力項目 | 必須 | 説明 |
   | --- | --- | --- |
   | 名前 | ✅ | 他の選手と重複しない名前 |
   | ふりがな | - | 並び替え・印刷時のふりがな |
   | ゼッケン番号 | - | 大会で使うゼッケン番号（数字以外もOK） |
   | 備考 | - | 運営用の自由記述メモ |
   | 参加大会 | - | チェックボックスから複数選択可能（0件でもOK） |

3. **「登録」** を押すと選手が追加されます。

## 編集

1. 一覧で対象の選手を **1人だけ** 選択します。
2. **「編集」** ボタンを押します（選手1人選択時のみ有効）。
3. ダイアログで項目を更新し、**「保存」** を押します。

:::info

参加大会の変更は **1人ずつ** 行います。複数選手の参加大会をまとめて変更する機能はありません。

:::

## 削除

- チェックボックスで **複数選択** した上で **「削除」** ボタン
- 確認ダイアログで選手名のリストが表示されます
- **「削除する」** で実行

:::caution

削除は取り消せません。関連するデータ（採点記録など）もすべて削除されます。

:::

## 備考欄の使い方

備考は自由記述できる運営用メモ欄です。採点画面には表示されないので、内部連絡用として活用できます。

- 例: 「アレルギー（卵）」「補助が必要」「保護者電話: 080-xxxx」「ロボット調整中」
- 検索ボックスで備考の内容からも選手を引けます

:::tip

同名の選手が複数いる場合は、ふりがなを活用して区別できます。ふりがなは印刷時の並び順にも使われます。

:::
