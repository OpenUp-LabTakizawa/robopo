# 採点者の登録

採点者（Judge）は、試合中に得点を記録する担当者です。ユーザー名・パスワードを持ち、**自分のアカウントでログイン** して担当大会の採点画面を使います。

:::tip[この画面を開く]

👉 [https://robopo.caravan-kidstec.com/judge](https://robopo.caravan-kidstec.com/judge)

:::

## 採点者一覧

<BaseImage src="/screens/03-admin/judge-list.webp" alt="採点者一覧" width={900} />

| 項目 | 説明 |
| --- | --- |
| ID | システムが自動採番する連番 |
| ユーザー名 | ログイン用のID（英小文字・数字・アンダースコアのみ） |
| 担当大会 | この採点者が採点を担当する大会（複数可） |
| ログイン日時 | 最後にログインした日時 |
| 備考 | 運営用の自由記述メモ |

## 検索・絞り込み・並び替え

- **検索ボックス**: ユーザー名・備考で検索
- **大会フィルタ**: プルダウンから1つの大会を選ぶと、その大会を担当する採点者だけを表示
- **並び替え**: ID または ユーザー名 で昇順／降順切替

## 新規作成

1. **「新規作成」** を押します。
2. ダイアログで以下を入力します。

   | 入力項目 | 必須 | 説明 |
   | --- | --- | --- |
   | ユーザー名 | ✅ | ログインに使うID。**英小文字・数字・アンダースコア（`_`）のみ** 使用可 |
   | パスワード | ✅ | **8文字以上** |
   | 備考 | - | 運営用メモ |
   | 担当大会 | - | チェックボックスから複数選択可能（0件でもOK） |

3. **「登録」** で採点者が追加されます。

:::caution[ユーザー名の制約]

ユーザー名は **英小文字（a-z）・数字（0-9）・アンダースコア（`_`）のみ** が使用できます。

- ✅ OK: `judge1`, `tanaka_t`, `center_court`
- ❌ NG: `Judge1`（大文字）, `田中`（日本語）, `judge-1`（ハイフン）, `judge 1`（空白）

入力欄では大文字を打っても自動的に小文字に変換されます。

:::

:::tip[パスワードは目のアイコンで確認できる]

入力中のパスワードは右端の 👁 アイコンで表示／非表示を切り替えられます。打ち間違いが心配な時に活用してください。

:::

## 編集

1. 一覧で対象の採点者を **1人だけ** 選択します。
2. **「編集」** を押します。
3. 変更したい項目を更新して **「保存」**。

編集時の注意:

- **ユーザー名**: 空欄にするとそのまま変更されません。変更したい場合だけ入力します
- **パスワード**: 空欄のままなら変更されません。新しいパスワードを設定したい時だけ入力します
- **担当大会**: チェックボックスで自由に増減可能

## 削除

- チェックボックスで **複数選択** → **「削除」** で一括削除できます
- 確認ダイアログで採点者名（ユーザー名）のリストが表示されます

:::caution

削除は取り消せません。採点者が記録した採点データなど関連データも同時に削除されます。

:::

## 採点者がログインすると

採点者アカウントでログインすると、**管理画面は表示されず、採点専用画面だけ** が表示されます。

- 自分が担当する大会のコースだけから採点を開始できます
- 採点結果には「誰が採点したか」が記録されるため、あとから採点者別の集計結果を確認できます

:::info

採点者はあらかじめ管理者がアカウントを作成してから、当日そのユーザー名・パスワードで採点者自身がログインします。

:::
