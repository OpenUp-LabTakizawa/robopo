# 採点の開始

この章では採点の開始方法について解説する。

## ダッシュボードからの採点開始

トップページはダッシュボード形式になっており、「採点」「集計結果」「大会管理」の3つのカードが表示される。

### 手順

1. **大会の選択**: アクティブな大会が1つだけの場合は自動選択される。複数ある場合はドロップダウンから選択する。
2. **採点者の選択**: 「採点者を選択」ドロップダウンから自分の名前を選ぶ。採点者を選択しないとコースへのリンクが無効になる。
3. **コースの選択**: 表示されたコースカードをタップして、選手選択画面に進む。

:::note
採点者の選択は必須です。選択されていない場合、コースカードはグレーアウトされ、タップできません。
:::

## 選手選択

コースを選択すると、選手一覧がカード形式で表示される。

- 各選手カードにはゼッケン番号と名前が表示される
- 選手数が多い場合は検索バーで絞り込みが可能
- 選手カードをタップすると直接採点画面に遷移する
