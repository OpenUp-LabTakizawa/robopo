# リトライ・コースアウト

採点中によく発生する「やり直し」「失敗」「コースアウト」の操作について解説します。

## 戻る（直前の判定を取り消す）

採点画面下の **「戻る」** ボタンは、**直前のミッション判定を取り消し、1つ前のミッションに戻す** 動作です。

- 間違って成功判定をしてしまった時
- ボタンを押した瞬間に失敗した時

に使います。戻った先のミッションを、もう一度正しく判定し直してください。

- **戻るボタンは最初のミッションでは使えません**（disabled）
- ゴール画面から戻ると、ゴール直前のミッションに戻れます

## 失敗

ミッションを **達成できなかった** 場合に **「失敗」** ボタンを押します。

- 結果送信ダイアログが開きます
- 失敗時点までの得点で結果を送信するか、**再挑戦** するか選べます

## コースアウト

ロボットがコースから外れてしまった場合に **「コースアウト」** ボタンを押します。

専用の **コースアウトダイアログ** が開き、コース設定に応じた得点計算が表示されます。

ダイアログには以下が表示されます。

- **1回目のスコア**（コースアウト時点の獲得ポイント）
- **コースアウト減点**（コースアウトルールが「減点」の場合のみ）
- 選択肢: **結果を送信して終了** / **再挑戦**（1回目のみ）/ **採点に戻る**

### コースアウト時の得点計算

コースエディタで設定したコースアウトルールに従って、その試行の得点が決まります。

| ルール | 得点計算 |
| --- | --- |
| 獲得済ポイント維持 | コースアウト時点の得点をそのまま記録 |
| 0点 | その試行は0点 |
| 減点 | 獲得済ポイント − 指定した減点 |

コースアウトルールは [コース作成](/docs/admin/course) 画面で設定します。

## 試行回数の上限（最大2回）

1人の選手は **最大2回** まで走行できます。

- 1回目（ゴール／失敗／コースアウト）の結果送信ダイアログで、**再挑戦** を選ぶと 2回目に進めます
- 2回目のダイアログでは **再挑戦** ボタンは表示されません
- 3回目以降を行いたい場合は、ホーム画面で選手を選び直す必要があります

:::info[2回走った場合の集計]

2回分の採点記録はデータベースに両方保存されます。集計時は **最高得点の試行** がその選手の成績として採用されます。

:::

## 確定した採点記録の修正

送信済みの採点記録を修正したい場合は、[集計結果](/docs/results/summary) → 選手名をクリックして **個人成績シート** を開き、**「編集」** モードで各試行の達成ミッション数を変更したり、試行を削除したりできます。

詳しくは [個人成績シート](/docs/results/score-sheet) を参照してください。

:::caution

当日の採点中は正確に操作することが最優先です。誤操作は **「戻る」** ボタンや **「採点に戻る」** リンクですぐに修正しましょう。

:::
