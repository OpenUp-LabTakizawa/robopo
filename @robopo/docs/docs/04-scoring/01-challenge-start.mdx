# 採点の開始

試合の採点は、トップページの **「採点」** カードから始めます。

:::tip[この画面を開く]

👉 [https://robopo.caravan-kidstec.com](https://robopo.caravan-kidstec.com)

:::

## 手順

### PCの場合

<BaseImage src="/screens/04-scoring/challenge-selector.webp" alt="採点開始セレクタ（PC）" width={800} />

### スマホの場合

2段組のレイアウトが縦スクロールに自動調整されます。

<BaseImage src="/screens/04-scoring/challenge-selector-mobile.webp" alt="採点開始セレクタ（スマホ）" width={360} />

:::info[採点者はスマホ運用が推奨]

採点は会場内で動きながら操作することが多いため、**タブレットやスマホを主な想定端末** として設計されています。各ボタン・リストは指でタップしやすいサイズに調整されています。

:::

採点画面に進むまでに、4つの項目をすべて選択する必要があります。

| 順序 | 項目 | 説明 |
| --- | --- | --- |
| 1 | 大会を選択 | **開催中** の大会のみ表示。開催中が1つだけなら自動選択 |
| 2 | コースを選択 | 選んだ大会に紐付き、かつ **設定済み** のコースのみ表示 |
| 3 | 採点者を選択 | 選んだ大会を担当する採点者のみ表示（検索可能） |
| 4 | 選手を選択 | 選んだ大会に参加する選手のみ表示（検索可能） |

すべて選ぶと **「採点を開始」** ボタンが有効になり、押すと採点画面に遷移します。

:::info[ログインの種類によって挙動が変わります]

- **採点者アカウント** でログイン中: 自分の名前が **採点者** として自動選択されます
- **管理者アカウント** でログイン中: 全採点者の中から手動で選ぶ必要があります

:::

:::tip[前回の選択を自動で覚えてくれます]

選んだ大会・コース・採点者は **ブラウザに保存** されており、次回同じ画面を開いた時に前回の選択が自動で復元されます。大会中の連続採点でも毎回選び直す必要がありません（選手だけは毎回選び直しになります）。

:::

## 選手の検索

選手欄の検索ボックスで、以下のいずれかに一致する選手を素早く絞り込めます。

- 名前
- ふりがな
- ゼッケン番号

検索ボックスは **大文字小文字を区別しません**。部分一致でOKです。

## 採点者の検索

採点者欄の検索ボックスで、**ユーザー名** の一部で絞り込めます。

## 選択できない時のメッセージ

各欄にはデータがない時に案内メッセージが出ます。

| 状況 | 表示 | 対応 |
| --- | --- | --- |
| 開催中の大会がない | 「開催中の大会がありません」 | 大会の日時設定を確認 |
| 大会を選択していない | 「大会を選択してください」 | まず大会を選ぶ |
| 大会にコースが割り当てられていない | 「コースが割り当てられていません」 | [大会一覧](/docs/admin/competition) でコースを紐付ける |
| 大会に採点者が割り当てられていない | 「採点者が割り当てられていません」 | [採点者一覧](/docs/admin/judge) で担当大会を紐付ける |
| 該当する採点者/選手が見つからない | 「該当する採点者（選手）がいません」 | 検索キーワードを変える |

## 採点を開始した後

**「採点を開始」** を押すと採点画面に遷移し、採点が始まります。

- 採点画面の戻る操作や離脱で、入力中の記録は **保存されません**
- ページ離脱しようとすると確認ダイアログが表示されます

採点画面の操作は [採点の仕方](/docs/scoring/scoring) を参照してください。
