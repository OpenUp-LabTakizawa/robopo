# 採点の仕方

採点画面の操作を説明します。採点UIはどのコースでも共通で、**コースで定義されたミッションが順に表示される** 仕組みです。

## 画面の見方

### PCの場合

<BaseImage src="/screens/04-scoring/scoring-basic.webp" alt="採点画面（PC）" width={900} />

### スマホの場合

同じ内容が縦長にレイアウトされ、フィールド図とボタン類が指でタップしやすいサイズに調整されます。

<BaseImage src="/screens/04-scoring/scoring-basic-mobile.webp" alt="採点画面（スマホ）" width={360} />

:::info[採点はスマホでの操作が想定されています]

会場内を歩きながら採点することが多いため、採点画面は **スマホ・タブレットでの操作を前提に最適化** されています。PCでも同じ機能が使えますが、当日の運用はタブレット＋立ち採点が推奨です。

:::

| 位置 | 内容 |
| --- | --- |
| 画面上部左 | **「1回目」/「2回目」** バッジ + コース名 / 選手名 |
| 画面上部右 | ミッション概観（現在 N / 全 M）・🔊 効果音切替・現在の合計点 **N pt** |
| 画面中央（非ゴール時） | 次に判定するミッションの内容 + フィールド図 |
| 画面中央（ゴール時） | 🎉 おめでとう表示 + ゴールボーナス + **「結果送信」** ボタン |
| 画面下部 | **「戻る」「コースアウト」「失敗」** ボタン |

画面上部にはミッション進行を示す **進捗バー** も表示されます。

## ミッション種類

コースに登録されているミッション種類に応じて、採点画面に表示される内容が変わります。ROBOPOで扱えるミッションは以下の5種類です。

| ミッション | 単位 | 内容 |
| --- | --- | --- |
| 前進 | パネル | 指定したパネル数だけ前に進む |
| 後進 | パネル | 指定したパネル数だけ後ろに下がる |
| 右回転 | 度 | 指定した角度だけ右に回る |
| 左回転 | 度 | 指定した角度だけ左に回る |
| 一時停止 | - | その場で停止（成功/失敗の判定のみ） |

## 採点の流れ

### PCの場合

<BaseImage src="/movies/02-scoring-flow.webp" alt="採点フロー（PC）" width={800} />

### スマホの場合

<BaseImage src="/movies/02-scoring-flow-mobile.webp" alt="採点フロー（スマホ）" width={360} />

1. **画面中央のフィールド図** に、ロボットの現在位置（●）と次に移動すべき目標パネル（▲ スタート印）が表示されます
2. ロボットが現在のミッションを達成したら、**目標パネルをタップ** — ポイントが加算され、次のミッションに進みます（効果音: ピコッ）
3. 失敗した場合は **「失敗」**、コースから外れた場合は **「コースアウト」** を押します
4. 戻って判定をやり直したい場合は **「戻る」** — 直前のミッション判定が取り消されます
5. 最後のミッションを達成すると **🎉 ゴール画面** に切り替わり、**「結果送信」** ボタンが表示されます
6. **「結果送信」** を押すと結果送信ダイアログが開きます

### 結果送信ダイアログ

ゴール時または失敗時に開くダイアログで、3つの選択肢があります。

| 選択肢 | 挙動 |
| --- | --- |
| **結果を送信して終了** | 今回の得点を確定してホーム画面に戻る |
| **再挑戦**（1回目終了時のみ） | 同じ選手でもう一度走行する（2回目）。得点は再挑戦の結果で上書きされる |
| **採点に戻る** | ダイアログを閉じて採点画面に戻る（戻るボタンで前のミッションに戻せる） |

## 試行回数

1人の選手は **最大2回** まで走行できます。画面上部の **「1回目」「2回目」** バッジが現在の試行回数を示します。

- 1回目がゴールまたは失敗で終了したとき、結果送信ダイアログに **再挑戦** ボタンが表示されます
- 再挑戦を選ぶとバッジが **「2回目」** に変わり、同じ選手で走行します
- 2回目の結果は1回目の結果を上書きして保存されます
- 3回目以降は選手選択画面からやり直す必要があります

:::info[集計時のルール]

保存された採点記録（1回目・2回目）のうち、**最高得点の試行** が選手の成績として集計されます。詳しくは [個人成績シート](/docs/results/score-sheet) を参照してください。

:::

## 段階評価ミッション

コース作成時に **段階評価** を設定したミッションでは、採点画面で複数の得点ボタンが表示されます。

- 画面中央のフィールド図の代わりに、**得点候補のボタンが縦に並んで** 表示されます
- 審判の判断で、どの得点にするか1つだけタップします
- 1番上の最高得点は緑、マイナス得点は赤で表示されます

:::tip[どんな時に使う？]

段階評価は「停止の精度に応じて10/5/3pt」など、結果の出来映えを段階的に評価したいミッションに使います。設定はコースエディタ側で行います。

:::

## コースアウトのルール

コースからロボットが外れた時の得点計算は、コースエディタで設定した **コースアウトルール** に従います。

| ルール | 挙動 |
| --- | --- |
| 獲得済ポイント維持 | コースアウトまでに得た点数を記録 |
| 0点 | その試行は0点扱い |
| 減点 | 獲得済ポイントから指定した点数を差し引く |

詳しくは [リトライ・コースアウト](/docs/scoring/retry-courseout) を参照してください。

## 効果音

採点画面では、操作に合わせて効果音が自動的に鳴ります。

| タイミング | 音 |
| --- | --- |
| 成功（次ミッションへ進む） | ピコッ（明るい通知音） |
| 戻る | ポン（低めの戻る音） |
| ゴール | ファンファーレ |

- 画面右上の **🔊 スピーカーアイコン** で ON / OFF を切り替え
- 静かにすべき会場では OFF、盛り上げたい場面では ON、と使い分けられます

:::tip

ゴール音は観客にも聞こえる大きさで鳴らすと、会場の一体感が出ます。会場の雰囲気やマイクの設置状況に合わせて調整してください。

:::
