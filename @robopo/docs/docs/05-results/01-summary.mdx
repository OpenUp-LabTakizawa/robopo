# 集計結果の見方

採点が終わったら、集計結果画面で成績を確認できます。選手・採点者・コースの3つの視点から集計データを見られます。

:::tip[この画面を開く]

👉 [https://robopo.caravan-kidstec.com/summary](https://robopo.caravan-kidstec.com/summary)

:::

<BaseImage src="/screens/05-results/summary.webp" alt="集計結果画面" width={1000} />

## 画面の構成

- **上部**: 大会セレクタ + タブ（選手 / 採点者 / コース）+ **一括印刷** ボタン
- **下部**: 選択したタブに応じた集計テーブル
- タブ内にも検索・絞り込み・並び替えツールがあります

## 大会の選択

画面上部の **大会セレクタ** で対象大会を切り替えられます。

- 表示されるのは **開催中 または 終了済** の大会のみ
- URL クエリ `?competitionId=N` で特定の大会を直接開くことも可能
- 自動選択ルール:
  1. 開催中の大会が1つあればそれが選ばれる
  2. なければ **終了日時が最も新しい** 終了済の大会
  3. それもなければ「大会を選んでください」表示

## 3つのタブ

### 選手タブ

選手ごとの採点成績を表示します。**コース単位** での集計なので、タブ内でさらにコースを切り替えて確認します。

| 列 | 説明 |
| --- | --- |
| 名前 | 選手名（クリックで [個人成績シート](/docs/results/score-sheet) へ） |
| ふりがな | ふりがな |
| ゼッケン | ゼッケン番号 |
| 初挑戦時刻 | このコースに最初に挑戦した時刻 |
| 完走時刻 | 初めて完走した時刻（コースアウト・失敗は除く） |
| 完走経過時間 | 初挑戦から完走までの経過時間 |
| 最終挑戦時刻 | 最後に挑戦した時刻 |
| 完走数 | 完走した試行の回数 |
| 初回得点 | 初回挑戦時の得点 |
| 最高得点 | 全試行のうち最も高かった得点 |
| 平均得点 | 全試行の得点の平均 |
| 総得点 | 得点合計（並び替え用） |
| 合計得点 | 全試行の得点を合算 |
| コースアウト数 | コースアウト扱いの試行数 |
| リトライ回数 | 再挑戦の回数 |
| 挑戦回数 | 全試行の回数 |

**検索**: 名前・ふりがな・ゼッケン番号で絞り込み
**並び替え**: 上記の多くの列で多段ソートが可能（複数キー組み合わせ）

### 採点者タブ

採点者ごとの採点活動を表示します。

- ユーザー名
- 採点人数
- 初採点時刻 / 最終採点時刻
- 採点回数
- 担当コース数
- 平均スコア
- コースアウト判定数

**検索**: ユーザー名で絞り込み

### コースタブ

コースごとの集計を表示します。

- コース名
- 初挑戦時刻 / 初完走時刻 / 最終挑戦時刻
- 挑戦者数・完走者数・完走率
- 総挑戦回数・平均スコア・最高スコア
- コースアウト数・リトライ数

**検索**: コース名で絞り込み

コース名をクリックすると、そのコースに挑戦した全選手の詳細が表示されます。

## 多段ソート（並び替え）

各タブの上部にあるソートツールバーで、**複数のキーを組み合わせた並び替え** ができます。

- 既定: 選手タブは「総得点（降順）」
- **キーを追加** で別のソートキーを追加
- キーごとに昇順／降順を切替
- キーを削除したり、すべてリセットしたりも可能

例: 「完走数（多い順） → 完走経過時間（短い順）」で並べると、完走実績と速さを両立した順位付けが可能。

## 一括印刷

画面右上の **「一括印刷」** ボタンから、選択中の大会の **全選手の個人成績シートをPDFで一括出力** できます。詳しくは [PDF出力](/docs/results/print) を参照してください。

:::info

集計結果は採点が完了するたびに自動で反映されます。会場でリアルタイムに順位を確認したい場合は、管理用の集計結果とは別に [観戦ページ](/docs/spectator/spectator) も活用してください（観戦ページはログイン不要）。

:::
