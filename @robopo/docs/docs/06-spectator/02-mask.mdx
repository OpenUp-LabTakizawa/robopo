# マスク機能について

大会終了間際に、観戦ページの **選手名を「???」に自動で隠す** 機能です。「誰が優勝するか分からない」というワクワク感を演出できます。

## この機能でできること

マスクが発動している間は、観戦ページの順位表で:

- **選手名が「???」** に置き換わります
- **ゼッケン番号が非表示** になります
- **得点と順位はそのまま表示** され続けます
- 画面上部に「**選手名は結果発表までシークレットです**」というバナーが表示されます

## 設定方法（管理者向け）

大会の作成または編集ダイアログで設定します。

1. [大会一覧](/docs/admin/competition) を開きます
2. 対象の大会の **チェックボックスを1つだけ選んで「編集」** を押します（新規作成時も同様に設定可）
3. ダイアログのコース選択の上にある **「終了間際に選手名をマスクする」** のチェックをONにします
4. チェックONにすると **「終了前 [N] 分からマスク」** の入力欄が現れるので、何分前から隠すかを入力します（1〜999分、既定値 30分）
5. **「保存」** で設定完了です

## マスクが発動する条件

次の3つをすべて満たしたとき、観戦ページのデータに自動的にマスクがかかります。

1. 大会の **「終了間際に選手名をマスクする」** 設定がON
2. 大会の **終了日時** が設定されている
3. **現在時刻 ≥ 終了日時 − マスク分** に達している

観戦ページはサーバーから **リアルタイムに更新** されるため、マスクON時刻に達すると次回の配信から自動で隠されます。

:::tip[いつ使うと効果的？]

- 表彰式直前に会場をざわつかせたい時
- 保護者参加のイベントで盛り上げたい時
- 動画配信中に「誰が勝つかまだ分かりません！」と煽りたい時

マスクOFFの大会では通常通り選手名が表示されます。イベントの性格に合わせて使い分けてください。

:::

## 状態ごとの表示

| 設定状態 | 観戦ページの選手名 | ゼッケン番号 |
| --- | --- | --- |
| マスクOFF | 常に表示 | 常に表示 |
| マスクON（終了の N 分より前） | 通常表示 | 通常表示 |
| マスクON（終了まで N 分以内） | **???** | 非表示 |
| 大会終了後 | **???** のまま | 非表示のまま |

:::info[大会終了後も隠されます]

大会の終了日時を過ぎても、設定を無効化しない限りマスクは継続されます。表彰式後に正式発表したい場合は、別途発表資料として [集計結果](/docs/results/summary) の [個人成績シート](/docs/results/score-sheet) や [PDF出力](/docs/results/print) を活用してください。

:::

## マスクを解除したい時

もう隠す必要がなくなったら、大会編集ダイアログで **「終了間際に選手名をマスクする」** のチェックを外して保存すれば、次回更新時から選手名が再び表示されます。
