# ライブランキングの見方

観戦ページは、会場にいる観客・保護者が **手元のスマホで順位を確認** できるライブランキング画面です。**ログインは不要** で、誰でもアクセスできます。

:::tip[この画面を開く]

👉 [https://robopo.caravan-kidstec.com/spectator](https://robopo.caravan-kidstec.com/spectator)

:::

<BaseImage src="/screens/06-spectator/leaderboard.webp" alt="観戦ページ" width={500} />

## 画面の構成

| 位置 | 内容 |
| --- | --- |
| 画面上部 | 🏆 **LIVE** バッジ + 「リアルタイムランキング」見出し |
| 大会表示 | 開催中の大会が **1つの場合**: 大会名をテキスト表示 / **複数ある場合**: セレクタから選択 |
| マスクバナー | マスク機能が発動中のみ表示（後述） |
| 更新インジケータ | 「N秒後に更新」のカウントダウン + 「今すぐ更新」ボタン |
| 進捗バー | 次回更新までの進捗を可視化 |
| ランキングカード | 順位・選手名・ゼッケン・得点・スコアバー |

## 順位の見方

- **1位**: 金色の枠 + 👑 王冠マーク、大きめのカード
- **2位**: 銀色の枠 + 🥈 メダルマーク、大きめのカード
- **3位**: 銅色の枠 + 🥉 メダルマーク
- **4位以下**: 白の標準カード
- 各カード内の **スコアバー** は1位の得点を100%としたときの比率

### 同点の扱い

得点が同じ選手は **同じ順位** が付きます（例: 1位が2人いると次は3位）。

### ランキングに表示される条件

- **得点が0ptより大きい** 選手のみ表示
- まだ1回も採点されていない選手や、0点しか取れていない選手は表示されません
- 初期状態やまだ採点が始まっていない時は「**まだ記録がありません**」と表示されます

## 自動更新

観戦ページは **15秒ごとに自動でデータを再取得** します。

- 画面上の「N秒後に更新」がカウントダウン
- 進捗バーが次回更新までの残り時間を表示
- 手動で更新したい場合は **「今すぐ更新」** ボタン

<BaseImage src="/movies/03-spectator-live.webp" alt="ライブ自動更新" width={420} />

## 大会の自動選択

ページを開いたとき、**開催中の大会** が自動選択されます。

- **開催中** = 大会の「開催日時」を過ぎていて、「終了日時」にまだ達していない
- 開催中が **複数ある場合** はセレクタが表示されるので手動で切り替えられます
- 開催中が **ゼロの場合** は「現在開催中の大会はありません」と表示されます

## 対応端末

- 観戦ページは **スマホ画面に最適化** されています
- PC・タブレットでも閲覧可能

:::info

ランキングは採点者の操作に連動しています。得点が反映されるまで、採点完了から最大15〜30秒程度かかる場合があります（キャッシュ・自動更新の間隔による）。

:::

## 選手名マスク機能

大会終了間際に選手名を自動的に「???」に隠すことができます。詳しくは [マスク機能について](/docs/spectator/mask) を参照してください。
