# ライブランキングの見方

観戦ページは、会場にいる観客・保護者が **手元のスマホで順位や挑戦の様子を確認** できるライブ画面です。**ログインは不要** で、誰でもアクセスできます。

:::tip[この画面を開く]

👉 [https://robopo.caravan-kidstec.com/spectator](https://robopo.caravan-kidstec.com/spectator)

:::

<BaseImage src="/screens/06-spectator/leaderboard.webp" alt="観戦ページ" width={500} />

## 画面の構成

| 位置 | 内容 |
| --- | --- |
| トップバー | 🔴 **LIVE** バッジ + 大会名 + 終了までの **残り時間カウントダウン** |
| マスクバナー | マスク機能が発動中のみ表示（後述） |
| メインパネル | 「**注目の挑戦**」直近のハイライト挑戦、または選んだ選手の **選手フォーカス** |
| ランキングパネル | 上位5名のリアルタイムランキング |
| コース最高記録 | コースごとの最高記録カード |
| 浮遊コントロール | ROBOPOロゴ（トップへ戻る）+ 接続インジケータ + 大会セレクタ |

## 注目の挑戦（メインパネル）

直近で **コース新記録** または **自己ベスト更新** が出た挑戦をハイライト表示します。

- 選手名・ゼッケン番号
- **得点**（カウントアップ演出付き）と最大得点
- そのコースの **コースマップ**（フィールド + ミッション + どこまで到達したか）
- **自己ベスト** / **挑戦回数** / **ステップ進行**（クリア時は「✓ クリア」表示）
- 該当時は **「コース新記録！」**（金色バッジ）または **「自己ベスト更新！」**（緑バッジ）

まだハイライト対象がない時は「**ハイライトはまだありません**」と表示されます。

## ランキングの見方

上位5名までを大きく表示します。

- **1位**: 金色のアクセント枠 + ハイライト背景
- **2位以下**: 標準の枠
- 各行に **順位・選手名・ゼッケン・挑戦回数・合計得点** が並びます

### 同点の扱い

得点が同じ選手は **同じ順位** が付きます（例: 1位が2人いると次は3位）。

### ランキングに表示される条件

- **得点が0ptより大きい** 選手のみ表示
- まだ1回も採点されていない選手や、0点しか取れていない選手は表示されません
- 初期状態やまだ採点が始まっていない時は「**記録待ち**」と表示されます

## 選手をタップして詳細を見る

ランキングの選手を **タップ（クリック）** すると、メインパネルがその選手の **「選手フォーカス」** 表示に切り替わります。

選手フォーカスでは、その選手の大会全体の成績が見られます。

- **総合順位**（得点が出ていれば表示）
- **ベストコース**: 一番得点が高いコースのコースマップ
- **コースごとの自己ベストと挑戦回数** の一覧
- **合計得点 / 合計挑戦回数**

### ほかの選手を探す

選手フォーカスの **「他の選手を探す」** ボタンを押すと、検索ダイアログが開きます。

- **名前・ふりがな** で絞り込み検索
- まだ挑戦していない選手も検索結果に出ます（順位はありません）
- 選択するとそのままその選手の選手フォーカスに切り替わります

選手フォーカスを閉じたい時は、右上の **「✕ 閉じる」** ボタンを押すと「注目の挑戦」表示に戻ります。

選んだ選手はブラウザに保存されるので、ページを再読み込みしても引き続き表示されます。

## ライブ演出（FX）

採点が反映されるたびに、画面上で **演出が流れます**。

| イベント | 演出 |
| --- | --- |
| 🏆 **コース新記録** | 大きなバナー + パーティクル + 画面フラッシュ |
| ⭐ **自己ベスト更新** | バナー + パーティクル |
| 🚀 **首位浮上**（ランキング1位の入れ替わり） | バナー（旧1位 → 新1位） |
| ▶ **通常の挑戦完了** | ティッカー（画面端を流れるテロップ）に追加 |

ティッカーには直近の挑戦履歴が流れ続けるので、何が起きているか一目で追えます。

## リアルタイム更新

観戦ページは **常時サーバーに接続** され、採点が登録されるたび **すぐに画面が更新** されます（ポーリングではなく、サーバーから即時通知される仕組みです）。

- 浮遊コントロール内の **小さな丸い接続インジケータ**
  - 🟢 **緑**: サーバーと接続中（リアルタイム更新が動いています）
  - 🔴 **赤**: 切断中（電波状況などで再接続を試みています）
- 通信状況によっては反映に数秒のずれが出ることがあります

## 大会の自動選択

ページを開いたとき、**開催中の大会** が自動選択されます。

- **開催中** = 大会の「開催日時」を過ぎていて、「終了日時」にまだ達していない
- 開催中が **複数ある場合** は浮遊コントロール内のセレクタで切り替えられます
- 開催中が **ゼロの場合** は「現在開催中の大会はありません」と表示されます

## 対応端末

- 観戦ページは **スマホ画面に最適化** されています（コンパクトレイアウト）
- PC・タブレットでも閲覧可能です（広めのレイアウトに自動で切り替わります）

:::info

ランキングや演出は採点者の操作と連動しています。会場の通信状況によっては、採点完了から画面に反映されるまで数秒かかる場合があります。

:::
