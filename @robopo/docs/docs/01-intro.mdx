# はじめに

ROBOPO（ロボポ）は、小学生向けロボットコンテスト **「キャラバンキッズテック」** の大会運営をサポートするWebアプリケーションです。

:::tip[本番環境]

👉 [https://robopo.caravan-kidstec.com](https://robopo.caravan-kidstec.com)

:::

## ROBOPOでできること

<BaseImage src="/screens/01-intro/home-guest.webp" alt="ROBOPOトップページ" width={800} />

ROBOPOは、大会運営の次のような作業をサポートします。

- **大会の管理**: 大会・コース・選手・採点者の登録と編集
- **コース作成**: 5×5マスのフィールドとミッションを組み合わせたコースを設計
- **採点**: タブレットやスマホで、競技中の得点を採点者がリアルタイムに記録
- **集計と印刷**: 選手ごとの成績を自動集計し、PDFで個別／一括出力
- **ライブランキング（観戦ページ）**: 会場の観客が手元のスマホで順位をリアルタイムに確認（ログイン不要）
- **選手名マスク**: 大会終了間際に観戦ページの選手名を自動で隠し、「誰が優勝するか分からない」ワクワク感を演出

## このマニュアルの読み方

このマニュアルは大会運営に携わる **管理者・採点者** を主な対象としています。観戦者の方向けの案内も含まれます。

| セクション | 対象 | 内容 |
| --- | --- | --- |
| 使い始める前に | 全員 | ログインの仕方・アカウントの種類 |
| 管理者向け | 管理者 | 大会・コース・選手の登録 |
| 採点者向け | 採点者 | 採点画面の操作方法 |
| 集計・印刷 | 管理者 | 成績の確認とPDF出力 |
| 観戦ページ | 観戦者 | ライブランキングの見方 |
| 技術情報 | 技術担当者 | インフラ・技術スタック |

## 対応端末

- **PC / タブレット / スマホ** のモダンブラウザ（Chrome / Safari / Edge 推奨）
- 採点はタブレット・スマホでの操作を想定しています
- 観戦ページはスマホ画面に最適化されています
