## 大会管理

### コース作成

作成したコースのリスト。削除や選択して編集する。
![playerpoint](/img/screen/12_courseList.webp)

3x3 のマス目でコースを表現する。「Start」「Route」「Goal」から選択して、マス目をクリックでパネルを置く。画面右側はロボットの動きを入れていくミッション設定。また、完成したコースとミッションで Goal にたどり着くかの有効性チェックを入れている。
![playerpoint](/img/screen/13_courseEditor.webp)

### 選手登録

名前、ふりがな(任意)、ゼッケン番号、QR コード(任意)を入力して登録できるようにしている。
![playerpoint](/img/screen/11_playerRegistermosaic.webp)

QR コードは選手登録の際に、QR を読み取って登録・採点できるようにしたいという意図があるが、現在は未実装。
