# 採点

この章では採点の仕方について解説する。

ルールやコースに関しては、ロボサバ公式サイトの[大会基本ルール](https://robosava.jp/competition/rule/basic-rule.html)が参考になる[^1]。

前章で各採点者に URL が発行されているので、基本的に各採点者はその URL(選手一覧)と採点中の画面しか使わないで済むようにできている。

:::note
以下を採点者に伝えておくとよい。

- トップページに行くボタンはあるため、もし選手一覧のページが分からなくなった場合は、発行した URL を開くこと。
- 結果を送信することでチャレンジ回数がカウントされるので、チャレンジをした場合は必ず結果を送信こと。
  :::
